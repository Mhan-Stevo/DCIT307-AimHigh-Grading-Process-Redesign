# Process Improvement Comparison

| Process Stage | Current System | Proposed System |
|---|---|---|
| Score Entry | Multiple Excel sheets | Single structured template |
| Calculations | Manual formulas | Automated calculations |
| Ranking | Manual ranking | Automatic ranking |
| Report Creation | Manual formatting | Auto-generated reports |
| Performance Analysis | Not available | Built-in analytics |
| File Management | Multiple files | Centralized template |

# Result

The proposed grading system reduces the grading process duration from **5 days to a few hours**.
