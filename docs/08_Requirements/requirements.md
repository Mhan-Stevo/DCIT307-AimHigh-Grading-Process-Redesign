# Functional Requirements

| ID | Requirement |
|---|---|

| FR2 | System must calculate totals automatically |
| FR3 | System must calculate averages |

| FR5 | System must generate report cards |

# Non-Functional Requirements

| ID | Requirement |
|---|---|
