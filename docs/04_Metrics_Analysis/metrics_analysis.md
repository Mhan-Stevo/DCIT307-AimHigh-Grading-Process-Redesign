# Metrics Analysis
## Aim High School Complex – Student Grading Process

---

# 1. Introduction

Metrics analysis evaluates the performance of the current grading and report generation system at Aim High School Complex.

The purpose of this analysis is to:

- Measure the efficiency of the current grading workflow
- Identify areas with performance issues
- Quantify delays and operational challenges
- Provide baseline data for improvement

These metrics will help compare the **current system (AS-IS)** with the **future improved system (TO-BE)**.

---

# 2. Key Performance Metrics

The following metrics were identified during interviews with teachers, administrative staff, and IT personnel.

| Metric | Description |
|------|-------------|
| Data Entry Error Rate | Frequency of errors when teachers input scores |
| Report Generation Time | Total time required to produce final reports |
| File Version Conflicts | Number of conflicting spreadsheet versions |
| Consolidation Time | Time required to combine subject results |
| Printing Time | Time needed to print report cards |

---

# 3. Current Performance Measurements

Based on observations and stakeholder interviews:

| Metric | Current Value |
|------|---------------|
| Data Entry Error Rate | ~70% of teachers experience errors |
| Report Generation Time | 4 – 5 days |
| File Version Conflicts | Frequent |
| Consolidation Time | 1 – 2 days |
| Printing Time | Several hours to overnight |

---

# 4. Performance Bottlenecks

### Manual Data Entry

Teachers manually input student scores into Excel sheets, which increases the likelihood of mistakes.

### File Version Conflicts

Multiple versions of grading sheets are often created, making it difficult to determine the correct data.

### Data Consolidation Delays

Administrative staff spend significant time merging subject files into a master grading sheet.

### Manual Ranking

Ranking students manually increases processing time and introduces errors.

### Printing Delays

Printing individual report cards requires manual effort and can take several hours.

---

# 5. Impact of Current Metrics

The current performance metrics lead to several operational issues:

- Delayed report distribution to parents
- Increased workload for administrative staff
- Higher likelihood of grading errors
- Lack of transparency and traceability

---

# 6. Summary

The metrics analysis confirms that the current grading system lacks efficiency and reliability.

Key issues include:

- High data entry error rates
- Slow report generation
- Poor file management
- Lack of automation

These findings justify the need for a redesigned grading system that introduces automation and centralized data management.
